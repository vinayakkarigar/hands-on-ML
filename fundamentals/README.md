# Machine Learning Fundamentals

This directory contains hands-on machine learning exercises and tutorials.

## Jupyter Notebooks

- [fundamentals\01_vectors.ipynb](01_vectors.ipynb) - vector properties

## Getting Started

1. Install required packages: `pip install -r requirements.txt`
2. Launch Jupyter: `jupyter notebook`
3. Open any notebook and run the cells

## Requirements

- Python 3.8+
- scikit-learn
- pandas
- numpy
- matplotlib
